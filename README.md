# Strategic Data Feeding for Semi-Supervised Domain Adaptation in Semantic Segmentation
This repository provides the code for the study "Strategic Data Feeding for Semi-Supervised Domain Adaptation in Semantic Segmentation".  
The DeepLabv3+ architecture with Resnet-101 backbone in this work is based on a repo published under the MIT license. It will be written here after the peer review.

## Datasets
Citsyscapes: https://www.cityscapes-dataset.com   
GTA5: https://download.visinf.tu-darmstadt.de/data/from_games  
SYNTHIA: https://synthia-dataset.net/downloads  

## Details
Requirements for the software versions are in the requirements.txt  
GTA5 --> Cityscapes files are under the gta-to-cs folder.  
SYNTHIA --> Cityscapes files are under the syn-to-cs folder.
